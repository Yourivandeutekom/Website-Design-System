Text-styles

Different text-styles for the heading elements and the paragraph element.

  • Heading 1 Element

    This heading is used to describe the title of the page.

    h1 {
      font-size: 2.2rem;
      font-weight: 700;
      font-family: 'Gantari', Arial, sans-serif;
    }

  • Heading 2 Element

    This heading is used to describe the subject of the section/article

    h2 {
      font-size: 1.7rem;
      font-weight: 700;
      font-family: 'Gantari', Arial, sans-serif;
    }

  • Heading 3 Element

    This heading is used to describe the subject of the paragraph text

    h3 {
      font-size: 1.3rem;
      font-weight: 700;
      font-family: 'Gantari', Arial, sans-serif;
    }

  • Paragraph Element

    Paragraph text, default text.

    p {
      font-size: 1rem;
      font-weight: 400;
      font-family: 'Gantari', Arial, sans-serif;
    }
